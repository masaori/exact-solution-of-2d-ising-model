# 可積分系

## 概要

十分な数の保存量（対称性）を持ち、原理的に厳密に解ける力学系・統計力学模型の総称。古典力学のLiouville可積分系から、量子群・Yang-Baxter方程式に基づく量子可積分系まで広がる。

## 2Dイジングモデルとの関連

- **Yang-Baxter方程式**: 2Dイジングモデルの転送行列が可換族をなす条件はYang-Baxter方程式に帰着する。これが厳密可解性の代数的な根拠
- **Onsagerの解法との関係**: Onsager (1944) の解法はLie代数（Onsager代数）の表現論に基づいており、これは量子可積分系の対称性代数の初期の例
- **Baxter のコミューティング転送行列**: 転送行列のパラメータ族が互いに可換であること（[T(u), T(v)] = 0）が、自由エネルギーの厳密計算を可能にする
- **逆散乱法との接点**: 連続極限で2Dイジングモデルは自由フェルミオン場に対応し、逆散乱法で扱える可積分場の理論と繋がる
