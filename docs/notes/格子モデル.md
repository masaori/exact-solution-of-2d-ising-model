# 格子モデル

## 概要

空間を離散的な格子で近似し、各格子点上の自由度（スピン、粒子数など）の相互作用を記述する統計力学のモデル群。

## 2Dイジングモデルとの関連

- **可解格子モデルの原型**: 2Dイジングモデルは Onsager (1944) によって最初に厳密に解かれた格子モデルであり、Yang-Baxter方程式に基づく可解模型の出発点となった
- **転送行列法**: 格子の行方向に沿って状態を積み上げる転送行列の固有値問題に帰着させる手法。本プロジェクトの中心的手法
- **他の可解模型との関係**: 6頂点模型、8頂点模型、Potts模型などへの一般化があり、イジングモデルは6頂点模型の特殊な場合と見なせる
- **相転移**: 2次元以上の格子モデルでは有限温度での相転移が起こりうる。イジングモデルはその最も単純な例
