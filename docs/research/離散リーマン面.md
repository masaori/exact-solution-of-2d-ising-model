# 離散リーマン面

## 概要

リーマン面（複素1次元多様体）の離散版。グラフ上に複素構造・正則関数・調和関数の類似物を定義し、連続極限でリーマン面の理論を回復する。

## 2Dイジングモデルとの関連

- **離散正則性**: Smirnov (2010) は臨界イジングモデルのフェルミオン相関関数が離散正則関数を満たすことを示し、これが共形不変性の厳密証明の鍵となった
- **連続極限との対応**: 格子間隔→0の極限で離散正則関数がリーマン面上の正則関数に収束し、臨界指数の普遍性が従う
- **一様化**: 離散リーマン面上の周期行列はヤコビ多様体の離散版を与え、イジングモデルの厳密解に現れる楕円関数・テータ関数との接点がある
