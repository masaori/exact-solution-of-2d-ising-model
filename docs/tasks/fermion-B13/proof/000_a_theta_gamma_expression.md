# a(θ_μ) の γ を使った表式

## 概要

`028_claim_a_theta_mu.typ` の証明を完成させ、フェルミオン定義 `029_definition_フェルミオン.typ` のnoteで参照されている `a(θ_μ)` の表式を γ₂(θ_μ), γ₂(-θ_μ) を用いて厳密に確定する。

最終的なゴールは以下の2つの等式を証明すること:

1. `a(θ_μ) = √(γ₂(θ_μ)/γ₂(-θ_μ))` （γを使った表式）
2. `a(θ_μ) = √((1 - α₁ e^{iθ})/(1 - α₁ e^{-iθ}) · (1 - α₂⁻¹ e^{iθ})/(1 - α₂⁻¹ e^{-iθ}))` （ホロノミック量子場の表式）

ここで α₁ = tanh(K₁)tanh(K₂*), α₂ = tanh(K₁)⁻¹ tanh(K₂*).

  - 証明の前半: `i√(γ₂γ₂(-))/γ₂(-)` を `±i·a(θ_μ)` に帰着する計算は arg の場合分けで進行中（L100〜L933）
  - 証明の後半: `√(γ₂(θ)/γ₂(-θ))` を α₁, α₂ の表式に変換する計算が R.1〜R.5 まで進行中（L934〜）だが未完

## スコープ

2. **後半: α₁, α₂ との接続**: γ₂(θ)/γ₂(-θ) を展開して α₁ = tanh(K₁)tanh(K₂*), α₂ = tanh(K₁)⁻¹ tanh(K₂*) による表式に変換する

## 作業内容

### 1. 前半の証明の整理（L100〜L933 相当）

028 の証明の前半では以下を示している:

```
i·√(γ₂(θ)γ₂(-θ)) / γ₂(-θ)
= cases(
    -i·√(γ₂(θ)/γ₂(-θ))  (arg ∈ (-π, 0]),
    +i·√(γ₂(θ)/γ₂(-θ))  (arg ∈ (0, π]),
  )
```

- この計算は arg^[0,2π) を用いた場合分けと `√(z₁z₂) = ±√(z₁)√(z₂)` の条件（039_claim）に依存している
- 現在の証明は非常に冗長なので、場合分けのロジックを整理し、各ステップに適切な #ref を付ける
- 特に `√(γ₂γ₂(-)) · √(1/γ₂(-)²)` → `±√(γ₂/γ₂(-))` の変換（L316〜L389）で使われる arg の範囲条件を明記する

### 2. 後半の証明: α₁, α₂ 表式への変換（L934〜 相当）

γ₂(θ_μ) の定義:
```
γ₂(θ) = i·e^{iθ}·s₂*·(c₁·cosθ - i·sinθ - s₁·c₂)
```

から出発し、

```
γ₂(θ)/γ₂(-θ)
= (e^{iθ}·(c₁·cosθ - i·sinθ - s₁·c₂)) / (e^{-iθ}·(c₁·cosθ + i·sinθ - s₁·c₂))
```

を変形して

```
= (1 - α₁·e^{iθ})(1 - α₂⁻¹·e^{iθ}) / ((1 - α₁·e^{-iθ})(1 - α₂⁻¹·e^{-iθ}))
```

となることを示す。

具体的手順:
- c₁ = cosh(2K₁), s₁ = sinh(2K₁), c₂ = cosh(2K₂), s₂* = sinh(2K₂*) を代入
- 分子分母を e^{±iθ} で整理
- tanh(K₁), tanh(K₂*) による因数分解で α₁, α₂ の積の形に変換
- R.1〜R.5 の計算を続行・完成させる

### 3. ステートメントの最終形の確認

上記1,2の結果をまとめて、028 のステートメント部分（L28〜L97）が正しいことを確認し、必要なら修正する。
