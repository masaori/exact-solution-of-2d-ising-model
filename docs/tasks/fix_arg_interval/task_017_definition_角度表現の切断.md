# Task: 017_definition_角度表現の切断.typ の修正

ステータス；完了

## 対象ファイル
`parts/000_計算公式/017_definition_角度表現の切断.typ`

## 変更概要
`s_((-pi, pi])` を `s_([0, 2pi))` に変更する。
これは角度の切断関数の定義変更であり、他の全てのファイルの基盤となる定義である。

## 変更のポイント

### 記号の変更
- `s_((-pi, pi])` → `s_([0, 2pi))`

### 区間と不等式の変更
旧: `-pi < theta - 2n pi <= pi`
新: `0 <= theta - 2n pi < 2pi`

**注意: 等号のつく場所が逆になる。**
- 旧: 左が開 `<`、右が閉 `<=`
- 新: 左が閉 `<=`、右が開 `<`

### 値域の変更
- `(-pi, pi]` → `[0, 2pi)`

## 具体的な変更内容

現在の定義:
```typst
$s_((-pi, pi]) : RR \/ ~_(angle) -> (-pi, pi]$ を以下のように定める。

$[theta]_(~_(angle)) in RR \/ ~_(angle)$ に対して、

$n in ZZ$ で、

$
  -pi < theta - 2n pi <= pi
$

を満たすようなものがただ一つ存在して、(証明略)

この $n$ を用いて、

$
  s_((-pi, pi])([theta]_(~_(angle))) := theta - 2n pi
$
```

変更後:
```typst
$s_([0, 2pi)) : RR \/ ~_(angle) -> [0, 2pi)$ を以下のように定める。

$[theta]_(~_(angle)) in RR \/ ~_(angle)$ に対して、

$n in ZZ$ で、

$
  0 <= theta - 2n pi < 2pi
$

を満たすようなものがただ一つ存在して、(証明略)

この $n$ を用いて、

$
  s_([0, 2pi))([theta]_(~_(angle))) := theta - 2n pi
$
```

## 注意事項
- 1行も省略しないこと。意図的に冗長に全ての計算ステップを明示している。
- このファイルは他の全てのarg関連ファイルの基盤となる定義なので、最初に修正すること。
