# Task: 018_definition_RRの角度表現.typ の修正

ステータス：完了

## 対象ファイル
`parts/000_計算公式/018_definition_RRの角度表現.typ`

## 変更概要
`s_((-pi, pi])` を `s_([0, 2pi))` に変更する。

## 変更のポイント

### 記号の変更
- `s_((-pi, pi])` → `s_([0, 2pi))`

### 区間と不等式の変更
旧: `-pi < ... <= pi` 系の不等式
新: `0 <= ... < 2pi` 系の不等式

**注意: 等号のつく場所が逆になる。**
- 旧: 左が開 `<`、右が閉 `<=`
- 新: 左が閉 `<=`、右が開 `<`

## 変更の参考例
`parts/000_計算公式/031_claim_複素数の積のarg.typ` で既に同様の変更が行われている。
このファイルでは旧版がコメントアウトされ、新版が `[0, 2pi)` の記法で記述されている。
変更前後を比較して、記法変更のパターンを理解すること。

### 特にこのファイルで注意すべき点
- `s_((-pi, pi])` の計算結果の具体的な値が変わる場合がある。
  例: `s_((-pi, pi])([- pi]_(~_(angle)))` は旧定義では `pi` だったが、新定義 `s_([0, 2pi))([- pi]_(~_(angle)))` では `pi` のまま（`-pi + 2pi = pi`, `0 <= pi < 2pi` を満たす）。
  ただし `s_((-pi, pi])([pi/2]_(~_(angle)))` は `pi/2` で変わらない。
- 具体的な数値での計算例がある場合、新しい区間 `[0, 2pi)` で再計算して正しいか確認すること。
