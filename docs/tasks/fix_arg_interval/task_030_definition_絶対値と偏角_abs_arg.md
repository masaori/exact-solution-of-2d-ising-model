`arg^((-pi, pi])` と `s_((-pi, pi])` を `arg^([0, 2pi))` と `s_([0, 2pi))` に変更する。

## 変更のポイント

### 記号の変更
- `arg^((-pi, pi])` → `arg^([0, 2pi))`
- `s_((-pi, pi])` → `s_([0, 2pi))`

### 定義の変更
旧:
```typst
$arg^((-pi, pi]): CC -> RR$ を
$
  arg^((-pi, pi])(z) := s_((-pi, pi])("pr"_2(phi_("polar")(z)))
$
と定め、zの偏角と呼ぶ
```

新:
```typst
$arg^([0, 2pi)): CC -> RR$ を
$
  arg^([0, 2pi))(z) := s_([0, 2pi))("pr"_2(phi_("polar")(z)))
$
と定め、zの偏角と呼ぶ
```
