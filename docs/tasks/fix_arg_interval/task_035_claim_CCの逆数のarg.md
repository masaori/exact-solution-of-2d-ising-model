# Task: 035_claim_CCの逆数のarg.typ の修正

ステータス：未完了

## 対象ファイル
`parts/000_計算公式/035_claim_CCの逆数のarg.typ`

## 変更概要
`arg^((-pi, pi])` と `s_((-pi, pi])` を `arg^([0, 2pi))` と `s_([0, 2pi))` に変更する。
$z^{-1}$ の偏角に関する claim とその証明を新しい区間で書き直す。

## 変更のポイント

### 記号の変更
- `arg^((-pi, pi])` → `arg^([0, 2pi))`
- `s_((-pi, pi])` → `s_([0, 2pi))`

### 不等式の変更
旧: `-pi < ... <= pi` 系
新: `0 <= ... < 2pi` 系

**注意: 等号のつく場所が逆になる。**

### 主張の分岐が変わる
旧 `(-pi, pi]` 版:
```
arg^((-pi, pi])(z^(-1)) = cases(
  -arg^((-pi, pi])(z)    (-pi < arg(z) < pi),
  pi                      (arg(z) = pi),
)
```

新 `[0, 2pi)` 版では:
- `arg^([0, 2pi))(z)` の値域は `[0, 2pi)` なので、`-theta` の値域は `(-2pi, 0]`
- `s_([0, 2pi))([-theta]_(~_(angle)))` は:
  - `theta = 0` のとき: `s_([0, 2pi))([0]) = 0`
  - `0 < theta < 2pi` のとき: `-theta + 2pi` が `[0, 2pi)` に入る → `2pi - arg(z)`

よって:
```
arg^([0, 2pi))(z^(-1)) = cases(
  0                            (arg^([0, 2pi))(z) = 0),
  2pi - arg^([0, 2pi))(z)     (0 < arg^([0, 2pi))(z) < 2pi),
)
```

### 証明の変更
- `s_([0, 2pi))` を使った展開に書き直す
- `[0, 2pi)` の区間で各ケースの不等式を再評価する

## 変更の参考例
`parts/000_計算公式/031_claim_複素数の積のarg.typ` で既に同様の変更が行われている。
このファイルでは:
- 旧版（3分岐）がコメントアウトされている
- 新版（2分岐）が `[0, 2pi)` の記法で記述されている

変更前後を比較して、以下を理解すること:
1. 記号 `(-pi, pi]` → `[0, 2pi)` の機械的な置換部分
2. 不等式の向きが逆になる（`<` と `<=` の入れ替え）
3. 分岐条件の再導出が必要な部分（値域の変更により cases が変わる）
4. 証明の各ステップでの不等式評価の変更

## 注意事項
- 1行も省略しないこと。意図的に冗長に全ての計算ステップを明示している。
- 証明の全行を書き直すこと。途中のステップを省略してはならない。
