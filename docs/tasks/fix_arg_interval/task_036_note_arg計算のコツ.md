`parts/000_計算公式/036_note_arg計算のコツ_極座標表現を使った偏角の計算方法.typ`

## 変更概要
`arg^((-pi, pi])` と `s_((-pi, pi])` を `arg^([0, 2pi))` と `s_([0, 2pi))` に変更する。
これはnote（解説メモ）なので、証明の厳密さよりも説明が新しい記法と整合することが重要。

## 変更のポイント

### 記号の変更
- `arg^((-pi, pi])` → `arg^([0, 2pi))`
- `s_((-pi, pi])` → `s_([0, 2pi))`

### 不等式の変更
旧: `-pi < ... <= pi` 系
新: `0 <= ... < 2pi` 系

**注意: 等号のつく場所が逆になる。**

### 計算例の変更
- 例に出てくる `arg^((-pi, pi])(z_1 z_2)` の計算手順を `arg^([0, 2pi))` 版に書き直す
- `s_((-pi, pi])` → `s_([0, 2pi))` の置換
- 不等式で値域を評価する部分を新しい区間で書き直す

## 変更の参考例
