# Task: 037_definition_CCのsqrt_複素数の平方根の定義.typ の修正

ステータス：完了

## 対象ファイル
`parts/000_計算公式/037_definition_CCのsqrt_複素数の平方根の定義.typ`

## 変更概要
`s_((-pi, pi])` を `s_([0, 2pi))` に変更する。
複素数の平方根の定義式中に `s_((-pi, pi])` が1箇所使われている。

## 変更のポイント

### 記号の変更
- `s_((-pi, pi])` → `s_([0, 2pi))`

### 定義式の変更
旧:
```typst
1/2 dot.op s_((-pi, pi])("pr"_2(phi_("polar")(z)))
```

新:
```typst
1/2 dot.op s_([0, 2pi))("pr"_2(phi_("polar")(z)))
```

### 注意
この変更により、$\sqrt{z}$ の定義自体が変わる。
- 旧: 偏角の半角を $(-\pi/2, \pi/2]$ の範囲で取る
- 新: 偏角の半角を $[0, \pi)$ の範囲で取る

つまり、branch cut の位置が変わる:
- 旧: 負の実軸で切断（$\arg = \pi$ で不連続）
- 新: 正の実軸で切断（$\arg = 0$ の近傍で注意が必要）

**この定義変更の影響が下流の全てのsqrt関連claimに波及するため、慎重に確認すること。**
