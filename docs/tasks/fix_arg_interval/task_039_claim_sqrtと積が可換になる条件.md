# Task: 039_claim_sqrtと積が可換になる条件_argの範囲による場合分け.typ の修正

ステータス：未完了

## 対象ファイル
`parts/000_計算公式/039_claim_sqrtと積が可換になる条件_argの範囲による場合分け.typ`

## 変更概要
`arg^((-pi, pi])` と `s_((-pi, pi])` を `arg^([0, 2pi))` と `s_([0, 2pi))` に変更する。
このファイルは51箇所の変更が必要で、最も変更量が多いファイルの一つ。

## 変更のポイント

### 記号の変更
- `arg^((-pi, pi])` → `arg^([0, 2pi))`
- `s_((-pi, pi])` → `s_([0, 2pi))`

### 不等式の変更
旧: `-pi < ... <= pi` 系
新: `0 <= ... < 2pi` 系

**注意: 等号のつく場所が逆になる。**

### 主張の分岐が変わる
旧 `(-pi, pi]` 版では3分岐:
```
-sqrt(z_1) sqrt(z_2)    (-2pi < arg(z_1) + arg(z_2) <= -pi)
sqrt(z_1) sqrt(z_2)     (-pi < arg(z_1) + arg(z_2) <= pi)
-sqrt(z_1) sqrt(z_2)    (pi < arg(z_1) + arg(z_2) <= 2pi)
```

新 `[0, 2pi)` 版では、`arg^([0, 2pi))(z_i)` の値域が `[0, 2pi)` になるため、和 `arg(z_1) + arg(z_2)` の値域は `[0, 4pi)` となる。
分岐は2つ（または新しい形の分岐）になるはず:
```
sqrt(z_1) sqrt(z_2)     (0 <= arg(z_1) + arg(z_2) < 2pi)
-sqrt(z_1) sqrt(z_2)    (2pi <= arg(z_1) + arg(z_2) < 4pi)
```

### 証明の変更
- 証明中の `s_((-pi, pi])` → `s_([0, 2pi))` の置換
- cos, sin の引数に含まれる `s_((-pi, pi])` の変更
- 三角関数の加法定理を使った展開部分は、引数が変わらない（`s` の結果が異なるだけ）
- 最終的な場合分けの不等式条件を再導出する

## 変更の参考例
`parts/000_計算公式/031_claim_複素数の積のarg.typ` で既に同様の変更が行われている。
このファイルでは:
- 旧版（3分岐）がコメントアウトされている
- 新版（2分岐）が `[0, 2pi)` の記法で記述されている

変更前後を比較して、以下を理解すること:
1. 記号 `(-pi, pi]` → `[0, 2pi)` の機械的な置換部分
2. 不等式の向きが逆になる（`<` と `<=` の入れ替え）
3. 分岐条件の再導出が必要な部分（値域の変更により cases が変わる）
4. 証明の各ステップでの不等式評価の変更
