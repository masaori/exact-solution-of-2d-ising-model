## 対象ファイル
`parts/000_計算公式/041_claim_自乗のsqrtとremark_負の実数の場合.typ`

## 変更概要
`arg^((-pi, pi])` を `arg^([0, 2pi))` に変更する。
$\sqrt{z^2}$ と $z$ の関係（正負の符号）に関する claim と remark を修正する。

## 変更のポイント

### 記号の変更
- `arg^((-pi, pi])` → `arg^([0, 2pi))`

### 不等式の変更
旧: `-pi < ... <= pi` 系
新: `0 <= ... < 2pi` 系

**注意: 等号のつく場所が逆になる。**

### 主張の分岐が変わる
旧 `(-pi, pi]` 版では3分岐:
```
-sqrt(z^2)    (-pi < arg(z) <= -pi/2)
sqrt(z^2)     (-pi/2 < arg(z) <= pi/2)
-sqrt(z^2)    (pi/2 < arg(z) <= pi)
```

新 `[0, 2pi)` 版では2分岐になる:
```
sqrt(z^2)     (0 <= arg^([0, 2pi))(z) < pi)
-sqrt(z^2)    (pi <= arg^([0, 2pi))(z) < 2pi)
```

### remark（負の実数の場合）の変更
- 旧: `arg^((-pi, pi])(z) = pi` の場合について議論
- 新: `arg^([0, 2pi))(z) = pi` の場合について議論（`pi` は `[0, 2pi)` の範囲内なので同様の議論が可能）

### 証明中の `#ref` について
- `#ref(<condition_of_commutativity_of_sqrt_and_product>)` → 039 を参照している
- 039 の修正結果に合わせて、引用している分岐条件を更新すること
