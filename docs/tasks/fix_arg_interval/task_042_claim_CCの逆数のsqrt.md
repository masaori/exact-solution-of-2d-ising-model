# Task: 042_claim_CCの逆数のsqrtとremark.typ の修正

ステータス：未完了

## 対象ファイル
`parts/000_計算公式/042_claim_CCの逆数のsqrtとremark.typ`

## 変更概要
`arg^((-pi, pi])` と `s_((-pi, pi])` を `arg^([0, 2pi))` と `s_([0, 2pi))` に変更する。
$\sqrt{1/z}$ に関する claim と remark を修正する。

## 変更のポイント

### 記号の変更
- `arg^((-pi, pi])` → `arg^([0, 2pi))`
- `s_((-pi, pi])` → `s_([0, 2pi))`

### 不等式の変更
旧: `-pi < ... <= pi` / `-pi < ... < pi` 系
新: `0 <= ... < 2pi` / `0 < ... < 2pi` 系

**注意: 等号のつく場所が逆になる。**

### 主張の分岐が変わる
旧 `(-pi, pi]` 版:
```
1/sqrt(z)      (-pi < arg(z) < pi)
-(1/sqrt(z))   (arg(z) = pi)
```

新 `[0, 2pi)` 版では:
- `arg^([0, 2pi))(z)` の値域は `[0, 2pi)`
- `arg^([0, 2pi))(1/z)` は 035 の結果を使う
- 分岐条件が新しい区間に合わせて変わる

### 証明の変更
- `s_([0, 2pi))([-theta]_(~_(angle)))` の計算を新しい区間で行う
- `arg(z) + arg(1/z)` の関係を新しい区間で再評価する

## 変更の参考例
`parts/000_計算公式/031_claim_複素数の積のarg.typ` で既に同様の変更が行われている。
このファイルでは:
- 旧版がコメントアウトされている
- 新版が `[0, 2pi)` の記法で記述されている

変更前後を比較して、以下を理解すること:
1. 記号 `(-pi, pi]` → `[0, 2pi)` の機械的な置換部分
2. 不等式の向きが逆になる（`<` と `<=` の入れ替え）
3. 分岐条件の再導出が必要な部分（値域の変更により cases が変わる）
4. 証明の各ステップでの不等式評価の変更
