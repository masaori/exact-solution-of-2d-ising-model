# P_μ の正規化定数の修正

## 概要

`027_claim_A_thetaの対角化_P_muとD_mu.typ` で P_μ を定義する際、固有ベクトルの自由定数 c を c=1 としている。この結果、`031_claim_psiの反交換関係.typ` のパート b) で

```
[ψ†_μ, ψ_ν]₊ = -4M|γ₂(θ_μ)|² · δ^M_{μ+ν,0}
```

となり、正準反交換関係 `δ^M_{μ+ν,0}` にならない。

c = 1/(2√M · γ₂(-θ_μ)) と選ぶことで正規化を修正する。

## 背景

- A(θ) の固有ベクトルは `v = c · (±i√(γ₂γ₂(-)), γ₂(-))^T` で、c は任意の非零複素数
- 026 ではこの自由度を残している
- 027 で P_μ を構成する際に c=1 と固定した結果、反交換関係に余分な係数が出る
- ホロノミック量子場 B.11/B.12 では適切な正規化が選ばれている

## 正規化後の定義

c = 1/(2√M · γ₂(-θ_μ)) とすると：

```
P_μ = 1/(2√M · γ₂(-θ_μ)) · ((+i√(γ₂γ₂(-)), -i√(γ₂γ₂(-))), (γ₂(-), γ₂(-)))
```

これにより、フェルミオンの定義は：

```
ψ̃†_μ = 1/(2√M) · (Ŷ_μ + i·a(θ_μ)·Ẑ^(-)_μ)
ψ̃_μ  = 1/(2√M) · (Ŷ_μ - i·a(θ_μ)·Ẑ^(-)_μ)
```
