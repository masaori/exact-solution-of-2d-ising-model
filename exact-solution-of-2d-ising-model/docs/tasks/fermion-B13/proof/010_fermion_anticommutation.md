## 作業内容

### 1. パート a) の完成: `[ψ_μ†, ψ_ν†]₊ = 0`

現在の計算の最終行（L389〜L413）は:

```
(-√(γ₂(θ_μ)γ₂(-θ_μ)))(√(γ₂(θ_ν)γ₂(-θ_ν)))
+ γ₂(-θ_μ)·γ₂(-θ_ν)
```

に `2Mδ^M_{μ+ν,0}` を掛けた形。δ^M_{μ+ν,0} ≠ 0 となるのは μ+ν ≡ 0 (mod M)、すなわち ν = -μ (mod M) のとき。

**ν = -μ のとき**:

```
γ₂(-θ_μ) = γ₂(θ_{-μ}) = γ₂(θ_ν)
```

なので、γ₂(-θ_ν) = γ₂(θ_μ) が成り立つ。これを代入すると:

```
-√(γ₂(θ_μ)γ₂(-θ_μ)) · √(γ₂(-θ_μ)γ₂(θ_μ))
+ γ₂(-θ_μ) · γ₂(θ_μ)
```

ここで `022_claim_gamma2_thetaとgamma2_minus_thetaの関係.typ` と `023_claim_gamma2_theta_muの積のarg.typ` の結果を使う:
- `arg^[0,2π)(γ₂(θ)γ₂(-θ)) = π` なので `γ₂(θ)γ₂(-θ) ∈ ℝ₋`
- よって `√(γ₂(θ)γ₂(-θ))² = γ₂(θ)γ₂(-θ)`（√の2乗は元に戻る、040参照）
- したがって `-√(γ₂γ₂(-))·√(γ₂(-)γ₂) + γ₂(-)γ₂ = -γ₂γ₂(-) + γ₂(-)γ₂ = 0`

上記のロジックを Typst の証明として記述する。

### 2. パート b): `[ψ_μ†, ψ_ν]₊ = δ_{μ≡ν (mod M)}`

ψ_μ† と ψ_ν の定義を展開:

```
ψ_μ† = i√(γ₂(θ_μ)γ₂(-θ_μ))·Ẑ_μ^(-) + γ₂(-θ_μ)·Ŷ_μ
ψ_ν  = -i√(γ₂(θ_ν)γ₂(-θ_ν))·Ẑ_ν^(-) + γ₂(-θ_ν)·Ŷ_ν
```

反交換子を展開すると4項になる:
- `+i√(γ₂γ₂(-))·(-i√(γ₂γ₂(-)))·{Ẑ_μ^(-), Ẑ_ν^(-)}₊` → `√(γ₂γ₂(-))²·2Mδ^M_{μ+ν,0}·I`
- `+i√(γ₂γ₂(-))·γ₂(-θ_ν)·{Ẑ_μ^(-), Ŷ_ν}₊` → 0
- `γ₂(-θ_μ)·(-i√(γ₂γ₂(-)))·{Ŷ_μ, Ẑ_ν^(-)}₊` → 0
- `γ₂(-θ_μ)·γ₂(-θ_ν)·{Ŷ_μ, Ŷ_ν}₊` → `γ₂(-)²·2Mδ^M_{μ+ν,0}·I`

{Ẑ^(-), Ŷ}₊ = 0 から中2項は消え、残りは:

```
(√(γ₂(θ_μ)γ₂(-θ_μ))·√(γ₂(θ_ν)γ₂(-θ_ν)) + γ₂(-θ_μ)γ₂(-θ_ν)) · 2Mδ^M_{μ+ν,0}
```

ν = -μ のとき、パート a) と同様の議論で `+γ₂γ₂(-) + γ₂(-)γ₂ = 2γ₂γ₂(-)` を得る。

ただし `2Mδ^M_{μ+ν,0}` と `γ₂γ₂(-)` の積で正規化を計算する必要がある。最終的に δ_{μ≡ν (mod M)} になることを確認する。

**注意**: 031 のステートメントでは `δ_{μ≡ν (mod M)}` になっているが、正規化定数が合うか要確認。ψの定義に入っている P_μ の係数によっては正規化が変わる可能性がある。

### 3. パート c): `[ψ_μ, ψ_ν]₊ = 0`

パート a) と完全に対称な計算。ψ_μ, ψ_ν を展開し、符号の違いから相殺することを示す。

### 4. 全体の整理

3つのパートを 031_claim_psiの反交換関係.typ に統合し、証明を完成させる。
