# T_(V)(フェルミオン) = exp(±γ)·フェルミオン の証明 (B.13)

## 概要

`030_claim_Vとpsiの交換関係.typ` で示された

```
T_(V)(ψ_μ†) = (γ₁(θ_μ) + √(-γ₂(θ_μ)γ₂(-θ_μ))) · ψ_μ†
T_(V)(ψ_μ)  = (γ₁(θ_μ) - √(-γ₂(θ_μ)γ₂(-θ_μ))) · ψ_μ
```

の固有値を `exp(±γ(θ_μ))` の形に書き直す。これにより `032_claim_Vprimeの表式.typ` の `V' = exp(-Σγ(θ_μ)(ψ†ψ - 1/2))` の正当化に至る。

## スコープ

以下の3ステップを含む新しい claim ファイル（または 032 の拡張）を作成する:

1. γ(θ_μ) の定義: cosh(γ(θ_μ)) = γ₁(θ_μ) を満たす γ ∈ ℝ の存在と一意性
2. sinh(γ(θ_μ)) = √(-γ₂(θ_μ)γ₂(-θ_μ)) であることの証明（cosh²-sinh²=1 の検証）
3. 固有値 = exp(±γ) であることの証明

## 作業内容

### 1. γ₁² + γ₂γ₂(-) = 1 の証明

030 の固有値 `γ₁ ± √(-γ₂γ₂(-))` が `exp(±γ)` と書けるための必要十分条件は:

```
(γ₁ + √(-γ₂γ₂(-))) · (γ₁ - √(-γ₂γ₂(-))) = 1
⟺ γ₁² - (-γ₂γ₂(-)) = 1
⟺ γ₁² + γ₂(θ)γ₂(-θ) = 1
```

これを直接計算で示す。

γ₁(θ) = c₁c₂* - s₁s₂*cosθ、γ₂(θ)γ₂(-θ) の具体形を代入して展開:

```
γ₁² = (c₁c₂* - s₁s₂*cosθ)²
    = c₁²c₂*² - 2c₁c₂*s₁s₂*cosθ + s₁²s₂*²cos²θ

γ₂(θ)γ₂(-θ) = [is₂*e^{iθ}(c₁cosθ - isinθ - s₁c₂)] · [is₂*e^{-iθ}(c₁cosθ + isinθ - s₁c₂)]
             = -s₂*² · [(c₁cosθ - s₁c₂)² + sin²θ]
             = -s₂*² · [c₁²cos²θ - 2c₁s₁c₂cosθ + s₁²c₂² + sin²θ]
```

よって:

```
γ₁² + γ₂γ₂(-)
= c₁²c₂*² - 2c₁c₂*s₁s₂*cosθ + s₁²s₂*²cos²θ
  - s₂*²c₁²cos²θ + 2s₂*²c₁s₁c₂cosθ - s₂*²s₁²c₂² - s₂*²sin²θ
```

ここで双対関係 `sinh(2K₂*)cosh(2K₂) = cosh(2K₂*)sinh(2K₂*) = ...`（Kramers-Wannier 双対性）:
- `s₂*c₂ = c₂*s₂*`（要確認: これは tanh(2K₂*) = tanh(2K₂) ではなく、双対関係 sinh(2K₂)sinh(2K₂*) = 1 から導かれる）
- `c₁² - s₁² = 1`, `c₂*² - s₂*² = 1` (双曲線関数の基本性質)

これらを用いて整理し、= 1 を示す。

**注意**: c₂* = cosh(2K₂*), s₂* = sinh(2K₂*) であり、Kramers-Wannier 双対関係 `sinh(2K₂)sinh(2K₂*) = 1` が鍵。この双対関係は既に `004_転送行列/000_definition_転送行列の記号の定義.typ` で定義されている K₂* の条件から導かれる。

### 2. γ(θ_μ) の定義

上記 1. が示されれば、`γ₁ = cosh(γ)` かつ `√(-γ₂γ₂(-)) = sinh(γ)` を満たす γ ∈ ℝ_{≥0} が一意に存在する。

- γ₁ ≥ 1 のとき: γ = acosh(γ₁)
- γ₁ < 1 のケースが物理的にありうるか検討（臨界温度付近）

新しい definition ファイル（例: `033_definition_gamma_theta_mu.typ`）を作成して γ(θ_μ) を定義する。

### 3. T_(V)(ψ) = exp(±γ)ψ の書き直し

030 の結果と Step 1, 2 を組み合わせて:

```
T_(V)(ψ_μ†) = cosh(γ(θ_μ)) + sinh(γ(θ_μ)) = exp(γ(θ_μ)) · ψ_μ†
T_(V)(ψ_μ)  = cosh(γ(θ_μ)) - sinh(γ(θ_μ)) = exp(-γ(θ_μ)) · ψ_μ
```

を明示的に記述する。

### 4. V' = exp(-Σγ(ψ†ψ - 1/2)) の正当化（032 の拡張）

Step 3 の結果と、010 で示したフェルミオン反交換関係を用いて、032_claim_Vprimeの表式.typ のステートメントを正当化する。

具体的には:
- ψ†ψ が各 μ ごとに独立な生成消滅演算子であること（010 の結果）
- exp(-γ(ψ†ψ - 1/2)) の ψ†, ψ への作用が exp(±γ)ψ†, exp(∓γ)ψ になること
- T_(V) と exp(-Σγ(ψ†ψ - 1/2)) が ψ†, ψ 上で同じ作用をすることから V' の形が定まること
